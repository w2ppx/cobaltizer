# cobaltizer
Automatically chooses fastest instances for you


## Installation

first fill out .env, example is avaliable in .env.example


you can install dependencies by using 

```shell
python -m pip install -r requirements.txt
```

start the script by
```shell
python3 main.py
```
yeah thats all
